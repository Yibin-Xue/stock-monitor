# Write a PRD Skill

This skill helps create comprehensive product requirement documents (PRDs) for new products or features. It includes:

- Product overview and objectives
- User stories and use cases
- Functional requirements
- Non-functional requirements
- UI/UX design considerations
- Development timeline
- Success metrics

When invoked, it will guide the user through the PRD creation process, asking relevant questions to gather necessary information.
