# A/B Test Setup Skill

This skill helps set up and configure A/B tests for websites, apps, or marketing campaigns. It includes:

- Test hypothesis formulation
- Test design and setup
- Sample size calculation
- Test duration determination
- Success metric definition
- Test implementation guidance
- Result analysis framework

When invoked, it will guide the user through the entire A/B testing process, from hypothesis creation to test implementation.
