# Onboarding CRO Skill

This skill helps optimize user onboarding processes to improve conversion rates and user retention. It includes:

- Onboarding flow analysis
- User journey mapping
- Conversion rate optimization strategies
- A/B testing for onboarding steps
- User feedback collection methods
- Onboarding metric tracking
- Best practices implementation

When invoked, it will help design and optimize onboarding experiences to maximize user engagement and conversion.
