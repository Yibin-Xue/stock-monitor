# Analytics Tracking Skill

This skill helps set up and configure analytics tracking systems for websites, apps, or digital products. It includes:

- Analytics platform selection and setup
- Event tracking configuration
- User journey mapping
- Conversion funnel analysis
- Custom dashboard creation
- Data visualization guidance
- Performance monitoring setup

When invoked, it will help implement comprehensive analytics tracking to measure user behavior and product performance.
