## 🤖 AI深度分析功能

### 使用方法
1. 在自选股列表中找到要分析的股票
2. 点击操作列的 "🤖 AI分析" 按钮
3. 等待10-30秒获取分析结果
4. 查看完整的分析报告

### 分析报告包含
- **投资建议**: 买入/增持/持有/减持/卖出
- **综合评分**: 0-100分
- **风险等级**: 低/中/高
- **技术分析**:
  - 短期趋势(MA5/MA10)
  - 中期趋势(MA20)
  - 长期趋势
  - 支撑位和压力位
  - RSI、MACD指标
- **基本面分析**:
  - 盈利能力(ROE、毛利率、净利率)
  - 偿债能力(资产负债率、流动比率)
  - 估值水平(PE、PB)
- **机会提示**: 当前市场机会
- **风险提示**: 潜在风险因素

### 示例
输入股票代码: **600519** (贵州茅台)
- 综合评分: 70分
- 投资建议: 增持
- 风险等级: 中
- 盈利能力: 优秀
- 偿债能力: 优秀
- 估值水平: 合理

## 💡 使用技巧

### 1. 快速分析多只股票
- 在自选股中依次点击"🤖 AI分析"
- 每只股票分析完成后自动保存到浏览器缓存
- 可以随时重新分析

### 2. 查看历史分析
- AI分析报告会在弹窗中显示
- 关闭弹窗后可以重新打开查看
- 建议保存重要的分析结果

### 3. 刷新行情数据
- 点击"刷新行情"按钮获取最新数据
- 刷新时间会显示在页面底部
- 建议每5-10分钟刷新一次

### 4. 添加自选股
- 点击"添加股票"按钮
- 输入6位股票代码
- 系统自动识别市场并添加

## 🐛 常见问题

### 1. 页面无法访问
**问题**: 打开 http://localhost:5173 显示错误

**解决**:
- 检查前端服务是否运行
- 确认端口5173未被占用
- 尝试重新启动前端服务

### 2. AI分析失败
**问题**: 点击"🤖 AI分析"后报错

**解决**:
- 检查后端服务是否运行
- 确认股票代码正确
- 查看浏览器控制台错误信息
- 查看后端终端日志

### 3. 数据不更新
**问题**: 行情数据不是最新的

**解决**:
- 点击"刷新行情"按钮
- 检查网络连接
- 确认数据API是否正常
- 可能是休市时间(周末或非交易时间)

### 4. 端口被占用
**问题**: 启动服务时提示端口被占用

**解决**:
```bash
# 查看占用端口的进程
netstat -ano | findstr :3002  # 后端端口
netstat -ano | findstr :5173  # 前端端口

# 杀掉进程
taskkill /PID <进程ID> /F
```
