## 🚀 快速开始

### 1. 后端启动
```bash
cd "d:/Trae CN/program/股票监控/backend"
npm install  # 首次运行需要安装依赖
npm start    # 启动后端服务 (端口3002)
```

### 2. 前端启动
```bash
cd "d:/Trae CN/program/股票监控"
npm install  # 首次运行需要安装依赖
npm run dev  # 启动前端开发服务器 (端口5173)
```

### 3. 访问应用
打开浏览器访问: http://localhost:5173
