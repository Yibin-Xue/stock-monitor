# 股票AI分析功能使用说明

## 功能概述

股票监控系统现已集成AI深度分析功能,可以为每只股票生成详细的投资分析报告,包括:

1. **技术分析** - 基于K线数据计算趋势、支撑压力位、RSI、MACD等指标
2. **基本面分析** - 评估盈利能力、偿债能力、估值水平
3. **投资建议** - 综合技术面和基本面,给出买入/增持/持有/减持/卖出建议
4. **机会风险提示** - 自动识别投资机会和潜在风险

## 技术架构

### 后端服务
- `backend/services/analysis_service.py` - Python分析服务,负责数据采集和计算
- `backend/routes/analysisRoutes.js` - Node.js路由,提供API接口

### 前端组件
- `src/components/StockAnalysis.jsx` - AI分析报告展示组件
- `src/components/StockAnalysis.css` - 样式文件
- `src/components/Watchlist.jsx` - 已集成AI分析按钮

## 使用方法

### 方法1: 在自选股列表中使用
1. 进入"自选股"页面
2. 在任意股票的操作列,点击"🤖 AI分析"按钮
3. 等待AI分析完成(约10-30秒)
4. 查看详细的分析报告

### 方法2: 直接调用API
```bash
# 分析单只股票
GET http://localhost:3002/api/analysis/{code}/analyze

# 批量分析(最多5只)
POST http://localhost:3002/api/analysis/batch-analyze
Body: {
  "codes": ["600519", "000001", "000002"]
}
```

### 方法3: 直接调用Python服务
```bash
cd backend/services
python analysis_service.py analyze 600519
```

## 分析报告内容

### 1. 股票基本信息
- 股票代码、名称
- 最新价格、涨跌幅
- 分析时间、数据来源

### 2. 投资建议
- **综合评分**: 0-100分,分数越高越看好
- **投资建议**: 买入/增持/持有/减持/卖出
- **风险等级**: 中低/中/中高/很高
- **短中长期**: 分别给出看好看谨慎建议

### 3. 技术分析
- **趋势判断**: 基于MA5/MA10/MA20的短中长期趋势
- **支撑压力位**: 基于最近30日的高低点
- **成交量趋势**: 放量/缩量/平稳
- **技术指标**: RSI(超买超卖)、MACD

### 4. 基本面分析
#### 盈利能力
- ROE(净资产收益率)
- 毛利率
- 净利率
- 综合评分与等级

#### 偿债能力
- 资产负债率
- 流动比率
- 综合评分与等级

#### 估值水平
- PE(市盈率)
- PB(市净率)
- 综合评分与等级(低估/合理/高估)

### 5. 机会与风险
- **投资机会**: 自动识别的看多因素
- **风险提示**: 自动识别的潜在风险

## 评分标准

### 投资建议分级
| 综合评分 | 建议操作 | 风险等级 |
|---------|---------|---------|
| 75-100 | 买入 | 中低 |
| 60-74 | 增持 | 中 |
| 45-59 | 持有 | 中高 |
| 30-44 | 减持 | 高 |
| 0-29 | 卖出 | 很高 |

### 技术面评分
- 短期趋势: MA5>MA10得80分,否则60分
- 中期趋势: MA10>MA20得75分,否则55分
- 长期趋势: 当前价>起始价得70分,否则50分

### 基本面评分
- 盈利能力(100分制):
  - 毛利率>40%: +30分
  - 净利率>15%: +30分
  - ROE>15%: +40分

- 偿债能力(100分制):
  - 资产负债率<30%: +50分
  - 流动比率>2: +30分

- 估值水平(100分制):
  - PE<15: +50分
  - PB<2: +30分

## 数据来源

- **实时行情**: 新浪财经接口
- **K线数据**: 专业金融数据API (CodeBuddy)
- **财务指标**: 专业金融数据API (CodeBuddy)
- **估值数据**: 专业金融数据API (CodeBuddy)

## 注意事项

1. **分析时间**: 完整分析需要10-30秒,请耐心等待
2. **数据时效性**: 财务数据基于最新财报,K线数据基于最近60个交易日
3. **风险提示**: AI分析仅供参考,不构成投资建议
4. **批量限制**: 批量分析最多支持5只股票,避免服务器过载
5. **价格处理**: 不同接口的价格单位可能不同,系统会自动处理

## 已知问题

1. **技术分析价格单位**: K线价格可能需要除法处理(如除以40),系统会自动判断
2. **财务指标字段名**: API返回字段名可能与标准名不同,已做映射处理
